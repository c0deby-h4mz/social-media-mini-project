 # Mini Social Media
Mini Social Media adalah aplikasi media sosial berbasis Command Line Interface (CLI) yang dibuat menggunakan Python.

## Fitur
- Registrasi dan Login User
- Edit Profil dan Bio
- Membuat Postingan
- Edit dan Hapus Postingan
- Like Postingan
- Follow dan Unfollow User
- Feed Postingan
- Pencarian Postingan
- Sistem Notifikasi Aktivitas

## Struktur Data yang Digunakan

### Linked List
Digunakan untuk menyimpan dan menampilkan feed postingan.

### Stack
Digunakan untuk menyimpan riwayat notifikasi aktivitas pengguna.

### Hash Map (Dictionary)
Digunakan sebagai cache data pengguna untuk mempercepat proses pencarian user.

## Algoritma
- CRUD (Create, Read, Update, Delete)
- Sorting berdasarkan waktu posting
- Searching berdasarkan kata kunci
- Hashing password menggunakan MD5

## Penyimpanan Data
Data disimpan menggunakan file CSV:
- users.csv
- posts.csv
- follows.csv

## Teknologi
- Python 3
- CSV Module
- Hashlib
- Datetime
